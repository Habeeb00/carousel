How to Use

Creating Carousels

1. Create New: Choose a resolution and number of frames, then click "Create Carousel" to generate a new carousel frame with visual guides.

2. Convert Existing: Select any frame in your design and click "Convert to Carousel" to add slide guides to it.

Working with Guides

Blue vertical lines mark slide boundaries

Slide numbers help you organize content

Guides are automatically hidden during export

Guidance layers are locked to prevent accidental edits

Exporting

Use the "Export Slides" button to create individual image slices for each slide. Then use Figma's File > Export to save them.
